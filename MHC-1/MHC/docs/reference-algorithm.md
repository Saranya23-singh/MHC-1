# MHC-1 Reference Algorithm (Section 2.1)

## System Algorithms

The MHC-1 (SootheSpace) system implements a **hybrid AI architecture** combining **supervised machine learning for stress/wellness prediction** and **generative NLP for conversational support**.

### 1. Stress/Wellness Prediction Model (Primary ML Algorithm)

- **Algorithm**: Random Forest Classifier (`sklearn.ensemble.RandomForestClassifier`, `n_estimators=100`)
- **Dataset**: `Sleep_health_and_lifestyle_dataset.csv`
  - **Features**: Sleep Duration, Quality of Sleep, Physical Activity Level, Heart Rate, Daily Steps, BMI Category (encoded 0-2)
  - **Target**: Stress Level (Low/Moderate/High via `pd.cut` + `LabelEncoder`)
- **Training Pipeline** (`ml_model/train_model.py`):
  ```
  1. Load CSV ‚Üí Preprocess (fillna mean, BMI encode)
  2. Train/Test split (80/20, random_state=42)
  3. Fit RandomForestClassifier ‚Üí Evaluate (accuracy_score, classification_report)
  4. Save: `joblib.dump(model, 'stress_model.pkl')`, `stress_le.pkl`
  ```
- **Inference** (`ml_model/predict.py` - Flask API `/predict`):
  - Load models ‚Üí Predict stress level + probabilities ‚Üí JSON response
- **Integration** (`server.js` `/api/predict-stress` ‚Üí proxy to Flask:5000):
  - `wellnessReport.js`: Fetches predictions ‚Üí Updates gauges, recommendations, PDF export
- **Fallback**: Rule-based (sleep_deficit + heart_excess + anxiety_score)
- **Accuracy**: Printed during training (e.g., `Accuracy: 0.XX`)
